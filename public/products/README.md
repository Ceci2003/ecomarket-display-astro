# Product Images Directory

Place all product images in this directory.

## How to use:

1. Add your product images here (e.g., `tea1.jpg`, `jam1.jpg`, etc.)
2. Reference them in `data/products.json` using paths like:
   - `/products/tea1.jpg`
   - `/products/jam1.jpg`

## Current product images needed:

Based on your `products.json`, you need these images:

- `tea1.jpg`, `tea2.jpg` - for "Балканска утрин" herbal tea
- `jam1.jpg`, `jam2.jpg` - for wild strawberry jam
- `mint1.jpg` - for dried mint
- `syrup1.jpg` - for peach syrup
- `candle1.jpg`, `candle2.jpg` - for soy wax candle
- `soap1.jpg` - for lavender soap

## Image recommendations:

- Format: JPG, PNG, or WebP
- Recommended size: 800x800px or larger (square format works best)
- File size: Keep under 500KB per image for fast loading
